# 前提知識とスキル要件

## 必須スキル

### 1. プログラミング言語

- **Python**

  - 基本文法
  - Type Hints
  - 非同期処理（async/await）
  - クラスとオブジェクト指向
  - デコレータ

- **TypeScript/JavaScript**
  - ES6+の機能
  - 非同期処理（Promise, async/await）
  - 型システム
  - モジュールシステム

### 2. 開発ツール

- **Git**

  - 基本的なコマンド
  - ブランチ戦略
  - PR の作成とレビュー

- **Docker**
  - コンテナの基本概念
  - Dockerfile
  - Docker Compose

### 3. データベース

- **PostgreSQL**

  - 基本的な SQL
  - リレーショナルデータベースの概念

- **Redis**
  - キャッシュの概念
  - 基本的な操作
